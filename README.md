# Dink Gear Shop

Dink Gear Shop is a pre-launch e-commerce brand selling custom pickleball apparel — 
tees and hats designed for players who want to rep the game in style. This repo is 
the full front-end website, built with pure HTML, CSS, and vanilla JavaScript. No 
frameworks, no build step. It's hosted on GitHub Pages at 
[dinkgearshop.com](https://dinkgearshop.com) via a custom GoDaddy domain.

The site includes a homepage with an email early-access signup, a shop page, an about 
page, a contact form, and an FAQ. All styling lives in a single shared `style.css` 
with a consistent design system — dark green palette, Poppins font, and CSS custom 
properties throughout. Product images are stored in `images/` as transparent-background 
PNGs colorized to match the brand's green theme.

To make changes, edit the HTML/CSS files directly and push to `main` — GitHub Pages 
deploys automatically.
