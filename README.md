# Dink Gear Shop

Dink Gear Shop is a pre-launch e-commerce brand selling custom pickleball apparel — 
tees and hats designed for players who want to rep the game in style. This repo is 
the full front-end + back-end website, built with pure HTML, CSS, and vanilla JavaScript. It's hosted on GitHub Pages at 
[dinkgearshop.com](https://dinkgearshop.com) via a custom GoDaddy domain.

The site includes following Front-end features: 
- Navigation Bar
- Email Signup
- Shopping features (Cart and checkout)
- About us
- FAQ page
- Home page
- Some animations

Backend Server features (Current work in progress):
- Database setup
- API development
- Order and product management functionality

To make changes, edit the HTML/CSS files directly and push to `main` — GitHub Pages 
deploys automatically.
